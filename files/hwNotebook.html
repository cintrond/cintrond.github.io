%load_ext rpy2.ipython

The code below provides an example of conducting a linear regression using both a frequentist approach and a bayesian approach in R. The data for the examples is simulated. pre is simulated using 1000 random draws from a uniform distribution with a minimum of 60 and maximum of 120. The pre variable can be thought of as a hypothetical pre-test. A data generating mechanism is arbitrarily chosen to simulate post (i.e. hypothetical post test scores which are a function of pre-test scores as well gender). A random covariate female is generated to represent gender. The proportion of females is set to 0.45. The coefficients in the model are arbitrarily chosen. The error or noise in the model is normally distributed with mean 0 and variance 1.

In [4]:
%R set.seed(1234567)
%R pre <- runif(n = 1000, min = 60, max = 120)
%R female <- rbinom(n = 1000, size = 1, prob = 0.45)
%R beta0 <- 100
%R beta1 <- 0.134
%R beta2 <- 7.13
%R post <- beta0 + beta1*pre + beta2*female + rnorm(n = 1000, 0, 1)
%R sim <- as.data.frame(cbind(pre,post,female))
%R plot(pre,post)
