Timeline of Models for the Structural Analysis of Moment Structures

  • Spearmans one factor model (1904)
  • Thurstones (1947) multiple-factor model
  • Anderson and Rubins (1956) restricted oblique common factor model
  • Bock & Bargmanns (1966) introduction of a group of models for the analysis of covariance structures
  • Jreskogs (1970) expansion of Bock & Bargmanns work to the general analysis of covariance structures (ACOVS)
    • Did not allow for a patterned invese matrix
  • Jreskogs (1973) linear structural relations (LISREL) model
    • Capitalized on work by Keesling and Wiley (Wiley, 1973) which showed that a patterned inverse was necessary for in a general-purpose model for multivariate analysis
  • Bentlers (1976) development of the m-th order factor model
    • Does not allow for a patterned inverse matrix
  • McDonalds (1978) covariance structural analysis model (COSAN)
    • m-th order model in which a patterned inverse matrix can be specified
    • ACOVS model when no patterned inverse matrix is necessary (McArdle, 2005)
  • McArdles (1978) Reticular Action Model
    • Second order model with an patterned inverse option

The Reticular Action Model (RAM)

  • A model for the structural analysis of moment structures
  • Designed to simplfy and unify structural equation models (McArdle, 2005)
    • COSAN and LISREL special cases & generalizations of RAM
  • Based on path analysis graphics
  • Mathematical Representation of RAM presentated as:
    • A second-order moment (i.e. \(\Sigma\)) structure model with two parameter matrices (\(\textrm{A}\) & \(\Omega\))
  • The A is represented in the RAM through a patterned inverse matrix
    • The term patterned here means that any element of a matrix can be defined with constants, paramaters to be freely estimated, or parameters which are constrained to be equal (McArdle, 2005; McArdle & McDonald, 1984)
  • Definitions and axioms outlined in McArdle (2005) provide isomorphism between RAM notation, graphics, and algebra

History RAM

  • McArdle (2005) provides detailed outline of RAM and its development
  • RAM rules initially presented between 1978-1981
  • McArdle used classical ideas and respective software related to linear structural relations (LISREL; Jreskog, 1973) and covariance structure analysis (COSAN; McDonald, 1978) to develop RAM
    • Also, used graphic concepts from latent variable path analysis (Wright, 1934; Wold, 1980)
  • The Reticular Action nomenclature was chosen to recognize work done by Raymond Cattell (1965,1978)
    • Cattell promoted a general system which included manifest and latent variances in a free-form network of aymmetric and symmetric action
    • Thefree-form network of asymmetric and symmetric action was termed the reticule
Handbook of Multivariate Experimental Psychology (Nesselroade & Cattell, 1966)

RAM

\[ \textrm{v = Av + u} \]

  • \(\textrm{v}\) is a t x 1 vector of random variables
  • \(\textrm{A}\) is a square matrix which represents the structural relationship (i.e.directed or asymmetric) between row variable \(\textrm{v}_{i}\) and column variable \(\textrm{v}_{j}\)
  • \(\textrm{u}\) represents the residual
  • Generalization of Guttmans (1953) image theory
    • \(\textrm{Av}\) are the images of t variables in terms of t-1 other variables
    • Components of \(\textrm{u}\) are the anti-images

\(\Omega\) Matrix

\[ \Omega = E[\textrm{uu}'] \]

  • t x t matrix
  • Each element of \(\Omega\), \(\omega_{ij}\), represents an undirected or symmetric structural relation among residual variables \(\textrm{u}_{i}\) and \(\textrm{u}_{j}\)
  • If no relationship among \(\textrm{u}_{i}\) and \(\textrm{u}_{j}\) \(\omega_{ij}\) = 0

\(\Sigma\) Matrix

  • In order to define the moments among the variables, in terms of structural parameters, the following t x t matrix must be defined as below
  • Here, every element of \(\Sigma\) represents the association among the variables \(\textrm{v}_{i}\) and \(\textrm{v}_{j}\)

\[\Sigma = E[\textrm{vv}'] \]

Moment Structure Matrix

  • \(\textrm{I}_{t}\) is a t x t identity matrix
  • We can re-write the RAM model as:

\[ \textrm{u = v - Av} = (\textrm{I}_{t} - \textrm{A)v} \]

  • As a result, we can write:

\[ \Omega = (\textrm{I}_{t} - \textrm{A})\Sigma(\textrm{I}_{t} - \textrm{A})' \]

  • Moreover, with the above definitions, we can now write the \(\textrm{v}\) variables in terms of the residual structural relations:

\[ \textrm{v} = (\textrm{I}_{t} - \textrm{A})^{-1}\textrm{u}\]

  • Which yields the moment structure model for all variables \(\textrm{v}\):

\[ \Sigma = (\textrm{I}_{t} - \textrm{A})^{-1}\Omega(\textrm{I}_{t} - \textrm{A})^{-1'} \]

  • \(\Sigma\) a function of A (i.e.patterned inverse) and \(\Omega\) (i.e.matrix of undirected or symmetric structural relations among residual variables u\(_{i}\) and u\(_{j}\)

Manifest and Latent Variables

In the RAM model, \(\textrm{v}\) is partitioned into two subvectors \(\textrm{m}\) of j components, and \(\ell\), of k components where j + k = t:

\[ \textrm{v}' = [\textrm{m}, \ell]' \]

  • \(\textrm{m}\) manifest variables
  • \(\ell\) latent or unobserved variables

Filter Matrix

  • \(_{j}\textrm{O}_{k}\) is a null matrix (i.e.all 0s)
  • In order to distinguish between manifest and latent variables, the following filter matrix is defined:

\[ F = [\textrm{I}_{j} : _{j}\textrm{O}_{k}]\]

Where \(_{j}\textrm{O}_{k}\) is a j x k matrix so that:

\[ \textrm{m = Fv} \]

  • F is a matrix of 0s and 1s that selects or filters the manifest variables out of v (i.e.the set of manifest and latent variables)
  • We will see later that in the RAMpath program F is not essentially to define as the program automates its construction based on model definitions
  • If positions of variables in v are permutated, that is the \(\textrm{m}\) and \(\ell\) are switched, the F matrix can correspondngly be permuted

Moment Structure Matrix for Manifest Variables

  • Let us define the moment structure matrix for manifest variables as:

\[ \Sigma_{mm} = E[\textrm{mm}'] \]

  • \(\Sigma_{mm}\) is a general raw product-moment matrix (note: \(\Sigma_{mm}\) is a covariance matrix if all manifest variables have means of zero).
  • Using \(\textrm{v} = (\textrm{I}_{t} - \textrm{A})^{-1}\textrm{u}\) and \(\textrm{m = Fv}\) we can define:

\[ \textrm{m} = \textrm{F}(\textrm{I}_{t} - \textrm{A})^{-1'}\textrm{u}\]

  • The moment sructure among manifest variables is then:

\[\Sigma_{mm} = \textrm{F}\Sigma\textrm{F}' = \textrm{F}(\textrm{I}_{t} - \textrm{A})^{-1}\Omega(\textrm{I}_{t} - \textrm{A})^{-1'}\textrm{F}'\]

Interrelationship Among the RAM and LISREL (McArdle, 2005)

  • Recall that the Linear Strucutral Relations (LISREL; Jreskog, 1973) can be defined using the following definitions:
    • \(x = \Lambda_{x}\xi + \delta\)
    • \(y = \Lambda_{y}\eta + \epsilon\)
    • \(\textrm{B}\eta = \Gamma\xi + \zeta\)
    • \(\textrm{x}\) (p x 1)
    • \(\textrm{y}\) (q x 1)
    • \(\Gamma\) (n x 1)
    • \(\xi\) (m x 1)
  • With expectations
    • \(\Sigma_{\zeta\zeta} = \Psi\)
    • \(\Sigma_{\epsilon\epsilon} = \Theta_{\epsilon}\)
    • \(\Sigma_{\delta\delta} = \Theta_{\delta}\)
    • \(\Sigma_{\xi\xi} = \Phi\)
  • McArdle (1984) showed, we can write the LISREL model in RAM notation as:

\[\begin{bmatrix}\textrm{y} \\\textrm{x} \\\end{bmatrix} = \begin{bmatrix}\textrm{I} & 0 & 0 & 0 \\0 & \textrm{I} & 0 & 0 \\\end{bmatrix}\begin{bmatrix}\textrm{I} & 0 & -\Lambda_{y} & 0 \\0 & \textrm{I} & 0 & -\Lambda{x} \\0 & 0 & \textrm{B} & -\Gamma \\0 & 0 & 0 & \textrm{I} \\\end{bmatrix}^{-1}\begin{bmatrix}\epsilon \\\delta \\\zeta \\\xi\\\end{bmatrix} \]

  • By defining the following:

    • \(j = p + q\),
    • \(k = n + m\),
    • \(\textrm{v}'= [y'x'\eta'\xi']\),
    • \(\textrm{u}'= [\epsilon'\delta'\zeta'\xi']\)

\[ \textrm{A} = \begin{bmatrix} 0 & 0 & \Lambda_{y} & 0 \\ 0 & 0 & 0 & \Lambda{x} \\ 0 & 0 & \textrm{(I-B)} & \Gamma \\ 0 & 0 & 0 & 0 \\ \end{bmatrix} \]

\[ \Omega = \begin{bmatrix} \Theta_{\epsilon\epsilon} & & & \\ 0 & \Theta_{\delta\delta} & & \\ 0 & 0 & \Psi & \\ 0 & 0 & 0 & \Phi \\ \end{bmatrix} \]

Where,

  • \(\Sigma_{\zeta\zeta} = \Psi\)
  • \(\Sigma_{\epsilon\epsilon} = \Theta_{\epsilon}\)
  • \(\Sigma_{\delta\delta} = \Theta_{\delta}\)

\[ \textrm{E} = (\textrm{I-A})^{-1} = \begin{bmatrix} \textrm{I} & 0 & \Lambda_{y}\textrm{B}^{-1} & \Lambda_{y}\textrm{B}^{-1}\Gamma \\ 0 & \textrm{I} & 0 & \Lambda_{x} \\ 0 & 0 & \textrm{B}^{-1} & \textrm{B}^{-1}\Gamma \\ 0 & 0 & 0 & 0 \\ \end{bmatrix} \]

\[ F = \begin{bmatrix} \textrm{I} & 0 & 0 & 0 \\ 0 & \textrm{I} & 0 & 0 \\ \end{bmatrix} \]

Decomposing Effects and Covariances for Mediation Model

  • RAMpath provides a flexible and efficient way to decompose each effect (direct or total effect) and covariance (variance or covariance) into individual components
  • Based on Wright (1934) tracing rules
  • Benefits
    • We can see the total effect between two variables
    • We can trace the variance of a variable to see what factors contribute to its variability
    • For any two variables, the covariance can be decomposed into individual components that contribute to the association between the two variables
# ramPathBridge with indirect=TRUE calculates all individual paths
bridge <- ramPathBridge(res1, allbridge=TRUE, indirect=TRUE)

Other Options for using RAM rules

  • RAM rules are not necessarily limited to any one program
    • The user, of the program, however, must be able to define:
      • a fixed F,
      • a patterned inverse of regression parameters \(\textrm{A}\), and
      • a patterned symmetric \(\Omega\) (McArdle, 2005)
  • RAMpath program
  • R-SEM package
  • OpenMx
  • SAS PROC CALIS with the use of the RAMLIST

Limitations & Benefits

  • Limitations
    • Sparse matrices can become unweildy
    • Majority of SEM literature relies on LISREL logic
  • Benefits
    • Economical representation of a number of models for the analysis of moment structures
    • Handles complex models in a straigtfoward manner
      • For ezample, nonlinear constraints on age, time, and cohort (McArdle, 2005)
    • Not limited to any specific SEM program
